Application Settings

Configure your attendance management application

The URL where the attendance server is running. Change this to connect to a different server.

Examples:
• http://127.0.0.1:6029 - Local server on this computer
• http://192.168.1.100:6029 - LAN server at specific IP
• http://server.company.com:6029 - Remote server
⚠️ Application Restart Required
Changing the server URL requires restarting the application to take effect.
